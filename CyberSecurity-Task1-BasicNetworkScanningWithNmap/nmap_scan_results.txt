4. Open Ports Explanation
========================================

Port: 135/tcp
Service: MSRPC (Microsoft Remote Procedure Call)

Purpose:
MSRPC is used by Microsoft Windows for communication between applications and services. It allows programs on one computer to request services from another computer on the network.

Security Risk:
If this port is exposed to untrusted networks, attackers may attempt to exploit Windows RPC vulnerabilities. It should be protected by a firewall and kept updated with security patches.

----------------------------------------

Port: 139/tcp
Service: NetBIOS-SSN

Purpose:
NetBIOS-SSN is used for Windows file and printer sharing. It enables computers on a local network to share files, folders, and printers.

Security Risk:
If improperly configured, attackers may gain access to shared files or collect information about the system. File sharing should only be enabled on trusted networks.

----------------------------------------

Port: 445/tcp
Service: Microsoft-DS (SMB)

Purpose:
Microsoft-DS uses the Server Message Block (SMB) protocol to provide file sharing, printer sharing, and other Windows network services.

Security Risk:
If SMB is outdated or not properly secured, it can be targeted by malware and network attacks. Keeping Windows updated and restricting SMB access helps reduce this risk.
